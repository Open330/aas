# Support

Use [GitHub Issues](https://github.com/Open330/aas/issues) for reproducible bugs, installation
problems, and focused feature requests. Include the aas version, platform/architecture, provider,
command, expected result, and sanitized output.

Before filing, check [README.md](README.md), provider-specific requirements, and the latest
[GitHub Release](https://github.com/Open330/aas/releases/latest). This project does not provide a
guaranteed response time or private account support.

Never post credentials or vault passphrases. Security vulnerabilities belong in the private
process described by [SECURITY.md](SECURITY.md).
