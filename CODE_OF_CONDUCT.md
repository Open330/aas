# Code of Conduct

We are committed to a welcoming, harassment-free project for everyone.

Be respectful, discuss technical decisions in good faith, accept constructive feedback, and keep
personal or sensitive information out of public reports. Harassment, discrimination, threats,
doxxing, and sustained disruptive behavior are not acceptable.

Maintainers may edit or remove contributions and restrict participation when behavior harms the
community. Report conduct concerns privately through the repository owner's GitHub profile. Do
not use security vulnerability reports for ordinary conduct disputes.

Enforcement decisions will consider context, impact, and a proportionate response while
protecting reporter privacy where possible.
